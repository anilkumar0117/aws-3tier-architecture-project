# AWS 3-Tier Architecture Project

## Project Overview

This project demonstrates deployment of a secure and scalable 3-tier architecture on AWS.

## Architecture

* Web Layer (EC2)
* Application Layer (EC2)
* Database Layer (Amazon RDS)

## AWS Services Used

* Amazon EC2
* Amazon RDS
* VPC
* Security Groups
* Internet Gateway
* Route Tables
* Load Balancer

## Features

* Public and private subnet setup
* Secure database deployment
* Scalable infrastructure
* Network isolation
